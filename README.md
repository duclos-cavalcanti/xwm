<h1 align="center">xwm</h1>
<p align="center">
  simple x window manager
</p>


Toy X-Window Manager to understand the X protocol.

## Installation
#### Ubuntu/Debian
```
sudo apt-get install libx11-dev
```

#### Arch Linux
```
sudo pacman -S libx11
```
